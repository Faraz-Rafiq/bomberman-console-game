# Bomberman Console Game 🎮

A console-based Bomberman-style game built in C++ as a Programming Fundamentals semester project.

## Features
- Grid-based movement with arrow keys
- Bomb placement and explosion logic
- Enemy AI with random movement
- Multi-stage progression
- High score tracking (file I/O)

## How to Run
- Open in Visual Studio
- Build and run `Project.cpp`
- Use arrow keys to move, `b` to place bomb, `q` to quit

## Tech
C++ | Windows Console API | File I/O
